Pasos para un algoritmo sencillo secuencial en Java:

  1. Crear un nuevo proyecto
  2. Dentro del proyecto crear un paquete, y dentro del paquete crear una clase
  3. Dentro de la clase crear un método main: public static void main (String [] args)
  4. Dentro del método main escribir el código a ejecutar
    Por ejemplo: System.out.println("Hola mundo");

Pasos para un algoitmo cualquiera con condiciones y repeticiones

  1. Hacer los mismos pasos de la lista anterior.
  2. Para una estructura condicional básica inicializamos una variable cualquiera. Por ejemplo int numero = 5
  3. Escribir if para inicial una estructura condicional. Por ejemplo:
    if (numero = 5) {
    System.out.println("El número es 5");
    }
  4. Ejecutar el código. Debería imprimir la cadena texto indicado
  5. Para una estructura repetitiva básica inicializamos una variable entera, con valr incial 0, que llamaremos contador: int cont = 0
  6. Escribir for para inicial un ciclo for. Por ejemplo:
    for (cont = 0; cont <=5; cont++) {
    System.out.println(cont);
    }
  7. Ejecutar el còdigo. Debería imprimir números del 0 al 5
