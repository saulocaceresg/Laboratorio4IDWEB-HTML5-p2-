Pasos para un algoritmo sencillo secuencial en Java:

  1. Crear un nuevo proyecto
  2. Dentro del proyecto crear un paquete, y dentro del paquete crear una clase
  3. Dentro de la clase crear un método main: public static void main (String [] args)
  4. Dentro del método main escribir el código a ejecutar
    Por ejemplo: System.out.println("Hola mundo");
